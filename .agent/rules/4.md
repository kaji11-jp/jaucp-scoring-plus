## 外部ライブラリのドキュメント

### DeepWiki MCP サーバー

GitHub リポジトリのドキュメントとコードベースの理解のために:

- **`ask_question` ツールを使用する** - 直接的な質問には、コード例を含む的を絞った回答が得られます
- **質問は具体的にする** - 「CSS テーマ設定システムはどのように機能しますか？」は、「このリポジトリについて教えてください」よりも優れています
- **効果的なクエリの例:**
- 「どのようなプラグインが利用可能で、どのように使用していますか？」
- 「HTML 構造を持つ基本的なプレゼンテーションをどのように作成しますか？」
- 「コントローラーやイベント処理を含むコアアーキテクチャは何ですか？」
